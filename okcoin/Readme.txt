Install the python package okcoinv5 from https://github.com/REAL-ERP/python-api-for-okcoin-V5
Download the OKC-Odoo module and place in the Odoo addon folder.
Obtain a V5 API key and secret from okcoin.com. Place the information and password in the auth.config file and update the path in the okcoin/models/__init__.py file in lines 11 and 12.
Update permissions to models and menus.
